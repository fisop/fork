# fork

Repositorio para el esqueleto del lab [fork](https://fisop.github.io/website/labs/fork) de la materia: **Sistemas Operativos - FIUBA**

## Compilar

```bash
make
```

## Pruebas

```bash
make test
```

O se pueden correr de forma nativa haciendo:

```bash
make test-native
```

## Linter

```bash
make format
```

Es necesario tener instalado `clang-format`. Basta con hacer:

```bash
sudo apt install clang-format
```
